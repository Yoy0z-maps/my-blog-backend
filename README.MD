### 명령어 설명

```
python manage.py runserver # 디버깅에서만 사용
python manage.py makemigrations # 새 모델 추가 혹은 모델 필드 변경 사항 있을 경우
python manage.py migrate # 위의 경우에 기존 마이그레이션을 데이터 베이스에 적용 혹은 샤ㅐ로운 데이터 베이스에서 기존 마이그레이션을 적용할 경우
```
